第二题：数学找规律

解题思路

  • 规律为：
    • 1：1
    • 2：1 + 3 + 1
    • 3：1 + 3 + 5 + 3 + 1
    • 4：1 + 3 + 5 + 7 + 5 + 3 + 1
    • ……
  • 于是，两个等差数列求和：(1 + 2 * n - 1) * n / 2 * 2 - 2 * n + 1 = n*n + (n - 1) * (n - 1)
