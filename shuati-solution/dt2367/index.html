给你一个下标从 0 开始、严格递增的整数数组 nums 和一个正整数 diff 。如果满足下述全部条件，则三元组 (i, j, k) 就是一个算术三元组：

  • i < j < k ，
  • nums[j] - nums[i] == diff 且
  • nums[k] - nums[j] == diff

返回不同 算术三元组 的数目。

解题思路

整体思路是枚举，不过本题条件很多，比如数组是严格递增的，数组长度、数组的每个元素的数值较小，所以有三种方法进行求解：

  1. 直接枚举每个元素，进行判断
