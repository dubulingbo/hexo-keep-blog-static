构建hexo博客页面

关于如何搭建hexo环境写博客，请参考hexo博客搭建教程

进入博客根目录，执行下述命令

  • hexo clean ：清理原来的构建痕迹，已便重新构建
  • hexo generate | hexo g ：生产静态页面，默认会在 <root>/public 目录下
  • hexo deploy ：部署网站
