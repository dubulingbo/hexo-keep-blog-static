1. 安装

1.1 环境准备

  • node环境 - 16.x

  • hexo安装

  • 配置hexo环境变量，将node_module目录加入path变量

  • 创建博客

1.2 hexo博客项目文件说明

  • _config.yml ：网站配置文件，用于设置站点信息，URL，部署，构建信息等
  • _config.theme名.yml ：主题配置文件，用于定制化个人主题
  • source ：源码文件夹
    • _data ：存放定制主题时相关静态资源
    • _post ：存放文章与草稿，markdown文件
  • theme ：主题文件夹，将需要更换的主题放入该目录下即可
  • db.json ：记录生成站点网页所需的数据
  • public ：静态网页站点生成目录，部署时也是将该目录下的所有文件拷贝的网站根目录下
